### Установка зависимостей

1. Windows
    ```bash
    git clone git@github.com:Swedist/study_itmo.git
    cd study_itmo
    python -m venv .venv
    .venv\Scripts\activate.bat
    pip install -r requerments.txt
    ```

2. MacOS
    ```bash
    git clone git@github.com:Swedist/study_itmo.git
    cd study_itmo
    python -m venv .venv
    source .venv/bin/activate
    pip install -r requerments.txt
    ```
